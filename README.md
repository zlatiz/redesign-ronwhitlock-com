# Ron Whitlock Reports

A modern NextJS 14 website for Ron Whitlock Reports - featuring roundtable discussions, interviews, and legislative updates from veteran Texas Broadcaster Ron Whitlock.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **NextJS 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for styling
- **Responsive Design** - looks great on mobile, tablet, and desktop
- **Multi-page routing** with dedicated pages for:
  - Home
  - Videos
  - Transcripts
  - Ron's Bio
  - Contact
  - Search
